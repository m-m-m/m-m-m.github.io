      • getOriginalValue

        V getOriginalValue()
        Returns:
        the original value since it was last set via setOriginalValue(Object) or most probably indirectly via setValue(Object). If this widget is modified, getValue() will return the current value as entered by the end-user while this method will still return the original value.
        ATTENTION: The original value shall never be modified to avoid undesired side effects. Various features such as resetValue() rely on this fact.

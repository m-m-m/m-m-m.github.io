Class BitMask

  • java.lang.Object
    • io.github.mmm.ui.api.datatype.bitmask.BitMask
  • Direct Known Subclasses:
    InverseBitMask, RegularBitMask
    public abstract class BitMask
extends Object
    An value and flag that can be encoded into a primitive int value. By creating multiple disjunct BitMasks you can encode multiple smaller informations ( boolean, Boolean, Enum, byte, etc.) into a single int value.

    1.0.0

      • isInverse

        public abstract boolean isInverse()
        Returns:
        true if the bits are used with inverse logic, so to set a flag according bits are set to 0 for true (instead of 1) and to 1 for false (instead of 0), false otherwise. The inverse logic is helpful to combine multiple InverseBitFlags with a logical AND in order to set an aggregated flag.

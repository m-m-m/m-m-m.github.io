Class UiSize

  • java.lang.Object
    • io.github.mmm.ui.api.datatype.UiSize
  • public class UiSize
extends Object
    UiSize represents a width or height of a UiWidget or the size of a font. Therefore it consists of two parts:
    • an amount
    • a unit
    Examples:
    toString() getAmount() getUnit()
    12px 12.0 UiSizeUnit.PIXEL
    100% 100.0 UiSizeUnit.PERCENT
    0.8em 0.8 UiSizeUnit.EM
    Since:
    1.0.0
    See Also:
    UiSizeUnit.newSize(double)

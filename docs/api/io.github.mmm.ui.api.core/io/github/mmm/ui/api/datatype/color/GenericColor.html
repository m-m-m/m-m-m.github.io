Class GenericColor

  • java.lang.Object
    • io.github.mmm.ui.api.datatype.color.GenericColor
  • public final class GenericColor
extends Object
    A Color based on factors.
    Note:
    Use Color for simple and efficient representation and transport of color information. However, if precision is required or for transformation between different color models use this class instead.
    Credits:
    The algorithms for transformation of the color models are mainly taken from HSL and HSV on wikipedia.
    ATTENTION:
    This implementation does not support color profiles or the Adobe RGB color space.
    Since:
    1.0.0

    public interface UiNativeWidget
extends UiWidget
    This is the marker interface for native UiWidgets that can be created from their interface.
    UiWidgets are defined as interfaces to abstract from the underlying implementation and native UI toolkit. In the source code those that can not be created directly are declared as abstract. As the abstract modifier has no further implication in Java this UiNativeWidget marker interface helps to specify which widget interfaces are provided as native implementations from the according toolkit specific UI implementations. All these interfaces except for UiNativeWidget itself can be created via UiWidgetFactoryNative.

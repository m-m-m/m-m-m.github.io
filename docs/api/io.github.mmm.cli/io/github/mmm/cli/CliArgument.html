    • Field Detail

      • END_OPTIONS

        public static final String END_OPTIONS
        The argument to indicate the end of the options. If this string is given as argument, any further arguments are treated as values. This allows to provide CliValues (e.g. a filename) starting with a hyphen ('-'). Please note that the first occurrence of this string given as argument will not be supplied as CliArgument but only parsed internally to hide complexity for the developer using this API.

      • getValue

        public String getValue()
        If this is a CliOption followed by a CliValue, this method will return that value (e.g. for "--key foo" it will return "foo"). If however such CliOption is followed by another CliOption or is the last CliArgument, this method will return null (e.g. for "-x -z"). Finally, if this is a CliValue, this method will return its value.
        Returns:
        the same as get() if this is a CliValue, otherwise if this a CliOption the value of the next argument if a CliValue or otherwise null.

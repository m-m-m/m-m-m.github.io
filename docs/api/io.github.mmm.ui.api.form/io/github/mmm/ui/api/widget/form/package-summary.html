    UiInput that acts as UiComposite to contain multiple UiInputs or other regular widgets.
    UiFormGroup<V>
    UiAbstractCompositeInput as native widget without value.

    UiMutableComposite for a form of UiInputs as children shown vertically from the top to the bottom.
    UiInputColumns
    UiCompositeInput that represents multiple UiInputs (or other regular widgets) as a single row with a label.
    UiInputContainer<V>
    UiRegularWidget for the container of a UiInput.
    UiInputRows
    UiCompositeInput that represents multiple rows of UiInputs.

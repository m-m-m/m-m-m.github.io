      • showPopup

        public void showPopup​(String message,
                      UiSeverity severity,
                      String title,
                      String details,
                      UiAction... actions)
        Description copied from interface: UiContext
        This method opens a popup window with the given message. It is the most generic and flexible but also the most inconvenient showPopup method variant.
        Parameters:
        message - is the message to display.
        severity - is the UiSeverity.
        title - is the title that will be displayed in the title-bar of the popup. May be null to use the default title for the given UiSeverity.
        details - the optional details. If not null details will be shown but are initially collapsed and have to be expanded by the end user.
        actions - are the UiActions for the buttons to close (answer, confirm, or cancel) the popup. Has to be at least one UiAction.

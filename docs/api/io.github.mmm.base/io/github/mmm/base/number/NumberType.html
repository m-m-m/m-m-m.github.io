Package io.github.mmm.base.number

Class NumberType<N extends Number>

  • java.lang.Object
    • io.github.mmm.base.number.NumberType<N>
  • Type Parameters:
    N - type of the represented number-class.
    public abstract class NumberType<N extends Number>
extends Object
    A NumberType represents a specific Number Class. It allows to check attributes like isDecimal() or getExactness().
    Further it acts as factory to create according numbers from string or from number.
    This is a class and NOT an Enum to be extensible.
    Since:
    1.0.0

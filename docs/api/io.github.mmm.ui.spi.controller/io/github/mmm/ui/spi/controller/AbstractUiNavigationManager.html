      • updatePlace

        public void updatePlace​(UiPlace place)
        Description copied from interface: UiNavigationManager
        Replaces the current place with the given UiPlace without creating a new entry in the navigation history.
        Specified by:
        updatePlace in interface UiNavigationManager
        Parameters:
        place - the updated UiPlace. Currently the ID of the current place may not change. This is only to update parameters. E.g. if you want to represent the selection of a master-detail dialog in the location (URL) you can use a parameter. This method allows to update the UiPlace and therefore the location without adding it to the history so UiNavigationManager.navigateBack() navigating back} is not replaying the entire history of the end-users selection what is typically undesired.

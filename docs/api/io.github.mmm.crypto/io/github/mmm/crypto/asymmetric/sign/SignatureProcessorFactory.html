    public interface SignatureProcessorFactory<S extends SignatureBinary,​PR extends PrivateKey,​PU extends PublicKey>
extends SignatureFactory<S>
    Interface for a factory to create instances of SignatureProcessor for asymmetric cryptography. Signatures only work with asymmetric security. For a given private key a SignatureSigner can be created that allows to sign any message. With the corresponding public key anyone can create a SignatureVerifier to verify the SignatureBinary.
    An instance of SignatureProcessorFactory typically combines asymmetric cryptography with hashing.

Interface UiMenu

  • All Superinterfaces:
    EventSource<UiEvent,​UiEventListener>, UiAbstractMenuItem, UiComposite<UiAbstractMenuItem>, UiDynamicComposite<UiAbstractMenuItem>, UiLabelledWidget, UiNativeWidget, UiWidget
    public interface UiMenu
extends UiDynamicComposite<UiAbstractMenuItem>, UiAbstractMenuItem, UiLabelledWidget, UiNativeWidget
    UiDynamicComposite representing a menu. A menu is something that is displayed by its label and opens a list of menu items if it is clicked. The user can click on one of these menu items to execute a specific functionality.
    A UiMenu is also a UiAbstractMenuItem so you can add it to another menu as a sub menu.

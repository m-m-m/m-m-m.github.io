Interface UiPopup

  • All Superinterfaces:
    EventSource<UiEvent,​UiEventListener>, UiAbstractWindow, UiClosableWidget, UiComposite<UiRegularWidget>, UiDynamicComposite<UiRegularWidget>, UiNativeWidget, UiWidget
    public interface UiPopup
extends UiAbstractWindow, UiNativeWidget
    UiAbstractWindow representing a modal popup window that blocks the client application until it is closed.
    NOTE:
    You typically want to call createAndAddCloseButton(). This is not done automatically so you can define the order of the buttons yourself.
    Since:
    1.0.0

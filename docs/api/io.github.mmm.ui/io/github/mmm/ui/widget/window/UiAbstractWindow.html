Module io.github.mmm.ui
Package io.github.mmm.ui.widget.window

Interface UiAbstractWindow

  • All Superinterfaces:
    EventSource<UiEvent,​UiEventListener>, UiClosableWidget, UiComposite<UiRegularWidget>, UiDynamicComposite<UiRegularWidget>, UiWidget
    All Known Subinterfaces:
    UiMainWindow, UiPopup, UiWindow
    public interface UiAbstractWindow
extends UiDynamicComposite<UiRegularWidget>, UiClosableWidget
    UiDynamicComposite that represents a window. To open the window use setVisible(true) and to close it use setVisible(false). Some underlying UI toolkits do not support to dynamically change the features of the window so configure them via setClosable(boolean), setResizable(boolean), etc. before opening the window.
    Since:
    1.0.0

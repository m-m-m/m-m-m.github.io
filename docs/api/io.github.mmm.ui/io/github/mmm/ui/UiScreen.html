Interface UiScreen

  • All Superinterfaces:
    AttributeReadSizeInPixel
    All Known Implementing Classes:
    AbstractUiScreen, FxScreen, TvmScreen
    public interface UiScreen
extends AttributeReadSizeInPixel
    This is the interface for the display where the UI objects are shown.
    ATTENTION:
    Be aware of multi-monitor setups. This object will represent the main screen OR the current screen where the UiMainWindow is located. This can depend on the underlying implementation or in case of a web-application on the browser.

    1.0.0

        Forces the display to update. Typically the screen-resolution does not change while your application is running. Therefore values of UiScreen may be pre-calculated and cached. However, you can call this method before reading resolution or screen type to force that you are getting most recent values.

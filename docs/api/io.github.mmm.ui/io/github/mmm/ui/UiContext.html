Module io.github.mmm.ui
Package io.github.mmm.ui

Interface UiContext

  • All Known Implementing Classes:
    AbstractUiContext, FxContext
    public interface UiContext
    Context and central API of the user-interface (UI). UIs consist of many smaller components that should be easy to create. Dependency injection (DI) would make the programming model more complex. In such cases you need to pass required components manually. This context is the central container to make this easy and maintainable. It acts as factory for other UiWidgets as well as contextual state for the UI.
    Since:
    1.0.0

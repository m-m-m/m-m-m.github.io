    Abstract base class for a Builder to create instances of a particular type of Property.
    As a Property typically has only one single required parameter that is also the most specific parameter, it is provided to the build-method. Therefore ProcessBuilder does not implement Builder and does not follow the default builder-pattern providing a non-arg build() method.
    For a specific Property type the builder should carry the same name with the suffix Builder. Example: BooleanProperty is build by BooleanPropertyBuilder.

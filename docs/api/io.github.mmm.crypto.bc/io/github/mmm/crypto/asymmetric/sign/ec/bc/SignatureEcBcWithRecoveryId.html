    Binary, Streamable, SignatureWithPublicKeyRecovery
    public class SignatureEcBcWithRecoveryId
extends SignatureEcBc
implements SignatureWithPublicKeyRecovery
    SignatureEcBc with an extra byte for the recovery ID. Allows to recover the public key. This enables to transfer only a compact hash of the public key to save capacity. E.g. BitCoin uses this to transmit a BitCoin Address instead of the actual public key.
    Since:
    1.0.0
